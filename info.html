Title Janiza Gesmundo Website
Goal By the end of this project, I want to produce a personal blog website to be shared to the public and be familiarized with Dreamweaver to create my own CSS website template.
Objective To be able to create and design my own CSS template using Dreamweaver and with the help of Photoshop and to add a lot of features in the website by using my knowledge in HTML and basic JavaScript.
Audience My audience will probably be everyone who shares the same interests as me. Teenagers in the society who like to read and are interested in articles and blogs that related to my hobbies and passion.
Description This website will be a personal blog containing stories, articles, videos, images and tutorials related to my hobbies and interests. This website can attract viewers who are interested in Computer Science, Programming, Web Design, Dance, Theater, Rock Bands, Music, Tourists in NYC, the Philippines and a lot more. I will also write articles and tutorials to share my knowledge about basic HTML, CSS, JavaScript, Java Programming and anything that is Computer Science related. I will blog about the East Coast Dance Community that will atract dancers in the society to know what are the latest news and events in the dance community in New York, New Jersey, DMV, Philly, Boston, etc. I will post dance vidoes of teams and choreography classes that may serve as a source for dancers to watch dance videos.
